# FleetFlow - Fleet & Logistics Management System

A comprehensive fleet and logistics management solution built with Next.js 16, TypeScript, and Tailwind CSS.

## Features

- **Dashboard** - Real-time fleet overview with key metrics
- **Vehicle Registry** - Complete vehicle management with CRUD operations
- **Trip Dispatcher** - Create, assign, and track trips with inline driver/vehicle creation
- **Maintenance Logs** - Track service history and scheduled maintenance
- **Expense Tracker** - Monitor fuel costs and operational expenses
- **Performance Analytics** - Driver performance metrics and safety scores
- **Analytics Dashboard** - Advanced reporting with charts and visualizations
- **Settings** - User preferences and system configuration
